nodejs安装
  1. 下载安装nodejs，去官网上面下载就可，下载好直接安装，不需要配置环境变量
  2. 下载完之后，Windows中在cmd(win+r)中使用命令：node,得到当前安装的node的版本
  3. 安装typescript
    npm install -g typescript
  4. 安装angular-cli
    npm install -g angular-cli@latest
